# 디자인 시스템

## 개요

- 서비스명: 티켓핀 (상품권 교환권 플랫폼)
- 플랫폼: 웹 (Next.js 16 + Tailwind CSS 4 + shadcn/ui)
- 기준 해상도: 모바일 우선 반응형 (320px ~ 1440px)

---

## 디자인 원칙

### 톤앤매너

**블랙 & 화이트 모노톤.** 불필요한 컬러를 배제하고 흑백 대비만으로 정보를 전달하는 미니멀 럭셔리 커머스. 넓은 여백과 깨끗한 타이포그래피로 고급스러운 인상을 만든다. 참고: 29CM, KREAM, 네이버 스마트스토어.

### 브랜드 키워드

- **프리미엄** — 블랙 & 화이트의 강한 대비로 만드는 고급감
- **명확** — 컬러 노이즈 없이 정보 구조만으로 전달하는 직관성
- **신뢰** — 결제/핀 번호를 다루는 서비스로서의 안정감
- **간결** — 이모지 없이, 최소한의 시각 요소로 완성하는 화면

### 참고 서비스

- **29CM** — 모노톤 기반 프리미엄 커머스, 넓은 여백, 에디토리얼 레이아웃
- **KREAM** — 블랙 & 화이트 중심, 깔끔한 상품 카드, 미니멀 UI
- **네이버 스마트스토어** — 상품 그리드, 카테고리 탭, 랭킹 UI
- **토스** — 깔끔한 폼/결제 UI, 상태 피드백 패턴

## 컬러 시스템

### Primary / Secondary

| 용도 | 변수명 | 값 | 설명 |
|------|--------|-----|------|
| Primary | `brand-primary` | `#000000` | 메인 CTA, 강조, 헤딩 |
| Primary (연한) | `brand-primary-soft` | `#F5F5F5` | 호버 배경, 선택 상태 |
| Primary (진한) | `brand-primary-dark` | `#000000` | 프레스/활성 (Primary와 동일) |
| Primary (매우 연한) | `brand-primary-muted` | `#FAFAFA` | 섹션 배경, 배너 배경 |
| Secondary | `brand-secondary` | `#171717` | 부제목, 보조 텍스트 강조 |
| Secondary (연한) | `brand-secondary-soft` | `#F5F5F5` | 보조 배경, 비활성 영역 |

> **Accent 컬러 없음.** 모노톤 일관성을 위해 별도의 포인트 컬러를 사용하지 않는다. 강조가 필요한 경우 블랙 배경 + 화이트 텍스트 반전, 또는 폰트 굵기/크기 차이로 해결한다.

### Semantic (상태 컬러)

모노톤 디자인이지만 상태 표현에는 최소한의 컬러를 사용한다. 컬러만으로 정보를 전달하지 않으며, 반드시 아이콘/텍스트를 병행한다.

## 둥글기 (Border Radius)

모노톤 럭셔리 테마에 맞게 둥글기를 절제. 지나친 라운드는 캐주얼해지므로 최소화.

| 토큰 | 값 | 용도 |
|------|-----|------|
| `radius-none` | 0 | 직각 요소, 풀 와이드 섹션 |
| `radius-sm` | 4px | 뱃지, 태그, 칩, 인라인 코드 |
| `radius-md` | 6px | 버튼, 인풋, 셀렉트, 드롭다운 |
| `radius-lg` | 10px | 카드, 토스트, 알림 |
| `radius-xl` | 14px | 모달, 바텀시트, 대형 카드 |
| `radius-2xl` | 18px | 히어로 섹션 카드, 프로모션 배너 |
| `radius-full` | 9999px | 아바타, 원형 아이콘 버튼 |

> `--radius` 기본값: `0.625rem` (10px). shadcn/ui의 sm/md/lg/xl은 이 값에서 파생.

## 그림자 (Shadow / Elevation)

모노톤 디자인에서 그림자는 은은하게. 블랙 기반이므로 opacity를 낮게 유지.

| 토큰 | 값 | 용도 |
|------|-----|------|
| `shadow-xs` | `0 1px 2px 0 rgb(0 0 0 / 0.04)` | 미세한 구분 (인풋 기본) |
| `shadow-sm` | `0 1px 3px 0 rgb(0 0 0 / 0.06), 0 1px 2px -1px rgb(0 0 0 / 0.06)` | 카드 기본, 드롭다운 |
| `shadow-md` | `0 4px 6px -1px rgb(0 0 0 / 0.06), 0 2px 4px -2px rgb(0 0 0 / 0.06)` | 호버 카드, 플로팅 버튼 |
| `shadow-lg` | `0 10px 15px -3px rgb(0 0 0 / 0.08), 0 4px 6px -4px rgb(0 0 0 / 0.06)` | 플로팅 요소, 팝오버 |
| `shadow-xl` | `0 20px 25px -5px rgb(0 0 0 / 0.08), 0 8px 10px -6px rgb(0 0 0 / 0.06)` | 모달, 다이얼로그 |
| `shadow-2xl` | `0 25px 50px -12px rgb(0 0 0 / 0.2)` | 최상위 오버레이 |
| `shadow-glow` | `0 0 20px rgb(0 0 0 / 0.12)` | 은은한 블랙 글로우 (CTA 호버) |

## 아이콘

| 항목 | 선택 |
|------|------|
| 라이브러리 | **Lucide React** (PRD 지정) |
| 스타일 | Outlined (기본), strokeWidth: 1.75 |

### 사이즈 규칙

| 토큰 | 크기 | 용도 |
|------|------|------|
| `icon-xs` | 14px | 인라인 뱃지 내부, 접미 아이콘 |
| `icon-sm` | 16px | 버튼 내부, 리스트 아이콘, 폼 아이콘 |
| `icon-md` | 20px | 기본 아이콘, 네비게이션 |
| `icon-lg` | 24px | 강조 아이콘, 카드 헤더 |
| `icon-xl` | 32px | 빈 상태, 상태 표시, 히어로 |
| `icon-2xl` | 48px | 대형 빈 상태 일러스트 대체 |

### 사용 가이드

- 아이콘과 텍스트 간격: `space-2` (8px)
- 아이콘 + 텍스트 정렬: vertical center (`items-center`)
- 단독 아이콘 버튼 시 `aria-label` 필수
- 이모지 사용 금지 — 모든 시각 표현은 Lucide 아이콘으로 대체
- 모노톤 테마에서 아이콘 색상은 `neutral-500` ~ `neutral-900` 범위

## 모션 / 애니메이션

Framer Motion 기반. 과도한 애니메이션 지양, 기능적 피드백 중심.

| 항목 | 값 |
|------|-----|
| 기본 트랜지션 | `200ms ease-out` |
| 빠른 트랜지션 | `150ms ease-out` (호버, 토글, 포커스) |
| 느린 트랜지션 | `300ms ease-in-out` (모달, 페이지 전환) |
| 페이지 전환 | Fade (opacity 0→1, 200ms) |
| 마이크로 인터랙션 | 버튼 호버: `scale(1.02)`, 프레스: `scale(0.98)` |
| 로딩 애니메이션 | 스켈레톤 UI (shimmer 효과, 펄스 금지) |
| 모션 원칙 | 기능적 피드백 우선. 장식적 애니메이션 최소화. `prefers-reduced-motion` 대응 필수 |

### 주요 모션 패턴

| 패턴 | 적용 | 설명 |
