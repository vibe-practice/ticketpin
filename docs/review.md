## 영역 8: 고객 플로우 E2E

### 플로우 A: 일반 구매 → 핀 확인

```
메인 → 상품 선택 → 결제(PG) → 바우처 링크(SMS) → 임시 비밀번호 입력
→ 사용자 비밀번호 설정 → 핀 번호 확인
```

### 플로우 B: 수수료 별도 구매 → 핀 확인

```
메인 → 상품 선택 → 결제(상품만) → 바우처 링크(SMS) → 임시 비밀번호 입력
→ 사용자 비밀번호 설정 → 수수료 결제(PG) → 핀 번호 확인
```

### 플로우 C-1: 구매 → 선물 → 수신자 핀 확인 (수수료 포함 상품)

```
구매자: 결제(상품+수수료) → 비밀번호 설정 → 선물하기 → 수신자 선택 → 선물 완료
수신자: SMS 수신 → 링크 클릭 → 임시 비밀번호 입력 → 비밀번호 설정 → 핀 번호 확인
```

### 플로우 C-2: 구매 → 선물 → 수신자 핀 확인 (수수료 별도 상품)

```
구매자: 결제(상품만) → 비밀번호 설정 → 선물하기 → 수신자 선택 → 선물 완료
수신자: SMS 수신 → 링크 클릭 → 임시 비밀번호 입력 → 비밀번호 설정
→ 수수료 결제(PG) → 핀 번호 확인
```

### 플로우 D: 구매 → 결제 취소

```
구매자: 결제 → (비밀번호 미설정) → 마이페이지 → 결제 취소
```

### 플로우 E: 관리자 결제 취소 (선물 후)

```
관리자: 주문 관리 → 선물된 주문 → 수신자 비밀번호 미설정 확인 → 결제 취소
```
